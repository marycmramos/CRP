# Knowledge Representation, Reasoning and Planning Systems

A collection of Artificial Intelligence projects exploring knowledge representation, logical reasoning, intelligent agents and automated planning. The work combines symbolic AI techniques, ontology-based reasoning and decision-making agents capable of operating in dynamic environments.

## Overview

This project explores multiple core areas of Artificial Intelligence through two complementary applications:

- Intelligent agents operating in a Pacman-inspired environment
- Automated resource management and scheduling using ontologies and planning

Together, these projects demonstrate how knowledge can be represented, reasoned upon and used to support intelligent decision-making.

---

## Project 1: Intelligent Agents for Dynamic Environments

A Pacman-inspired environment featuring autonomous ghost agents capable of perception, reasoning and strategic decision-making.

### Implemented Agents

#### Propositional Logic Agent
- Maintains a knowledge base of observed information
- Detects and tracks targets
- Uses logical reasoning to guide decisions

#### Patrol and Interception Agent
- Combines patrol strategies with target pursuit
- Dynamically switches behavior according to observations
- Uses pathfinding for efficient navigation

#### Strategic Agent
- Stores facts about the environment
- Predicts future movements
- Performs strategic interception

### Topics Explored

- Intelligent Agents
- Knowledge Representation
- Propositional Logic
- First-Order Logic
- Search Algorithms
- Multi-Agent Systems

## Project 2: Ontology-Based Resource Management and Automated Planning

A symbolic AI system designed to manage academic resources and automatically generate valid examination schedules.

### Ontology-Based Management

An ontology was developed to model:

- Students
- Teachers
- Courses
- Rooms
- Equipment
- Activities

Reasoning mechanisms allow validation of constraints and intelligent resource allocation.

### Automated Planning

Exam scheduling was formulated as a planning problem using PDDL.

The planner automatically generates schedules while respecting:

- Room availability
- Capacity constraints
- Course conflicts
- Resource requirements

### Topics Explored

- Ontologies (OWL)
- Automated Planning
- PDDL
- Constraint Satisfaction
- Resource Optimization
- Symbolic AI
